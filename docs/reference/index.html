Getting started

Install globally:

npm install -g @lamnhan/ngxer

Init the project:

ngxer init

Or, using npx:

npx @lamnhan/ngxer init

Edit .ngxerrc.json, then generate content:

ngxer generate

Command overview

Tool for prerendering Angular apps

  • ngxer generate|g
  • ngxer init|i [projectPath]
  • ngxer remove|x <inputs...>
  • ngxer report|r --detail
  • ngxer update|u <inputs...>
  • ngxer help
  • ngxer *

Command reference

generate

Generate static content.

Usage:

ngxer generate
ngxer g

init

Add ngxer to a project.

Usage:

ngxer init [projectPath]
ngxer i [projectPath]

Parameters:

  • [projectPath]: Custom path to the project

remove

Remove a generated content.

Usage:

ngxer remove <inputs...>
ngxer x <inputs...>

Parameters:

  • <inputs...>: List of inputs to be removed

report

Show generated statistics.

Usage:

ngxer report --detail
ngxer r --detail

Options:

  • -d, --detail: Show detail.

update

Update a static.

Usage:

ngxer update <inputs...>
ngxer u <inputs...>

Parameters:

  • <inputs...>: List of inputs to be updated

help

Display help.

Usage:

ngxer help

*

Any other command is not suppoted.

Usage:

ngxer <cmd>
